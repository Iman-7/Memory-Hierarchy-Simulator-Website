Welcome to our Memory Hierarchy Simulator!!

This project is used to implement a Memory Hierarchy Simulator to trace the Direct Mapping caching system.

  • To use the simulator, please go to the "Test" tab to enter the input.
  • After enetering all the inputs, you will be immediately redirected to the "Output" tab where you can see the output.
  • After each memory access and until the sequence ends, the program outputs the following:
    1) Valid bits and tags of all entries in the cache represented in a table.
    2) Total number of accesses.
    3) The hit and miss ratios.
    4) The Average Memory Access Time (AMAT) of the memory hierarchy (in cycles).
